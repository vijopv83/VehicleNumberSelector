# 🚗 Vehicle Number Selector

A smart web application that helps users discover **meaningful, memorable vehicle numbers** using intelligent filters like lucky numbers, numerology, patterns, prime sequences, Fibonacci, zodiac alignment, and more.

Ideal for vehicle buyers, dealerships, premium number bidders, and numerology enthusiasts.

---

## 🚀 Features

### 🔎 Smartcarry Number Discovery
Generate curated vehicle numbers using:

- Lucky numbers  
- Numerology totals  
- Prime numbers  
- Fibonacci sequence  
- Palindromes  
- Repeating digits  
- Ascending/descending patterns  
- Zodiac alignment  
- Birthdate compatibility  

---

### 🏆 Top 8 Intelligent Suggestions
AI-ranked top 8 numbers based on:

- Meaning score  
- Symmetry  
- Memorability index  
- Pattern strength  

---

### 🎯 Custom Ending Search

Search preferred endings (e.g., `786`, `999`, `007`) with match options:

- Exact match  
- Contains pattern  
- Starts with  
- Ends with  

---

### 📊 Side-by-Side Comparison

Compare shortlisted numbers using:

- Meaning score  
- Symmetry rating  
- Recall value  
- Cultural relevance  
- Pattern uniqueness  

---

### 🌟 Lucky & Zodiac Filters

Personalized filtering options:

- Personal lucky number  
- Zodiac compatibility  
- Birthdate alignment  
- Name numerology  
- Avoid unlucky combinations  

---

### 🧠 Pattern Intelligence Engine

Automatically detects:

- Mirror numbers  
- Double pairs (1122)  
- Triple repeats (1111)  
- Prime endings  
- Fibonacci fragments  
- Symmetrical layouts  

---

### 📈 Explore All Results

Sort and filter by:

- Highest numerology score  
- Most symmetrical  
- Most rare  
- Most memorable  

Includes pagination or infinite scroll support.

### 📱 Responsive UI

- Mobile-first design
